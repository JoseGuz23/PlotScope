# PlotScope Landing Page

This is the corporate landing page for PlotScope, focusing on the company and its flagship product, LYA.

## Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

## Build

To build for production:

```bash
npm run build
```

## Structure

- `src/App.jsx`: Main landing page component.
- `src/index.css`: Stylesheet using Tailwind CSS, matching the LYA product theme.
